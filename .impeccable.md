## Design Context

### Users
Solo investor/researcher scanning for asymmetric investment opportunities across 8 categories. Uses the dashboard to review, filter, compare scans over time, and track favorites. Power user — values density but also clarity.

### Brand Personality
Refined, confident, intelligent.

### Aesthetic Direction
**Refined Dark / Luxury** — inspired by Linear, Carta, Mercury Bank. Elegant, clean, generous spacing. Premium typography. NOT a terminal or Bloomberg clone. Feels like a private research tool built for one person.

**Anti-references**: Generic AI dashboards with cyan-on-dark, purple gradients, glassmorphism. Bloomberg-style density without purpose. Generic shadcn/ui templates.

**Theme**: Dark mode primary, light mode available via toggle.

### Design Principles
1. **Calm density** — Show lots of data without visual noise. Use spacing and hierarchy, not decoration.
2. **Typography carries the design** — Font choice and weight variation create hierarchy. Minimal use of color badges.
3. **Warm, not cold** — Warm-tinted neutrals (amber hint), not blue/gray. Gold accent. Feels expensive.
4. **Functional color only** — Green = positive, red = negative, amber = attention. Never decorative.
5. **One typeface, many weights** — Plus Jakarta Sans across the board. No monospace except data/tickers.
